# dash

## clone repository

```bash
git clone git@github.com:kazupyong/dash.git
cd dash
```

## install nodebrew 

```bash
curl -L git.io/nodebrew | perl - setup 
nodebrew install v8.1.3
nodebrew use v8.1.3
```

## npm install

```bash
npm install
```

## edit .envfile

```bash
cp env.sample .env
vi .env
```

## start node server

```bash
npm start
```
